🛡️ Audit-AI Pro: Forensic Fraud Detection Engine
Hybrid AI for Indian Accounting Standards
An adaptive, auto-trainable forensic tool designed for Chartered Accountants to detect financial anomalies, TDS splitting, and manual ledger manipulation.

🚀 Key Features
Hybrid Intelligence: Combines XGBoost (Supervised) and Isolation Forest (Unsupervised).
Auto-Trainable Pipeline: Integrated feedback loop allowing the model to learn from new audits.
Forensic Visuals: Benford's Law analysis and Risk Probability timelines.
Black Shine UI: Modern, high-contrast interface with glass-morphism effects.
🏗️ Architecture
The system is divided into an Inference Engine (app.py) and an Autonomous Trainer (trainer_module.py) to ensure stable, continuous learning.
